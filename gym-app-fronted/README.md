# 🏋️ IronPeak Gym Ｗeb App
# Overview
The app is designed to handle the full user-facing experience of a gym management system, including:

# Features
1. Role-based access control for Trainers and Members with distinct workflows.
2. Class management — create, edit, delete, and schedule recurring classes with level and capacity settings.
3. Enrollment system — join classes, join waitlists, and get automatically promoted when spots open.
4. Attendance tracking — trainers can mark member attendance per session.
5. Calendar view — monthly class schedule for both trainers and members.
6. Profile management — update email, change password, and view personal class history.
7. Real-time feedback — toast notifications, inline validation, and live capacity indicators.
8. Advanced filtering — search by keyword, filter by level, and sort by name or difficulty.
9. Modular architecture — context-based state management ready to connect to any REST or GraphQL backend.
