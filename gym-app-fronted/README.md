# 🏋️ IronPeak Gym Ｗeb App
# Overview
# The app is designed to handle the full user-facing experience of a gym management system, including:

# Role-based access control for Trainers and Members with distinct workflows.
# Class management — create, edit, delete, and schedule recurring classes with level and capacity settings.
# Enrollment system — join classes, join waitlists, and get automatically promoted when spots open.
# Attendance tracking — trainers can mark member attendance per session.
# Calendar view — monthly class schedule for both trainers and members.
# Profile management — update email, change password, and view personal class history.
# Real-time feedback — toast notifications, inline validation, and live capacity indicators.
# Advanced filtering — search by keyword, filter by level, and sort by name or difficulty.
# Modular architecture — context-based state management ready to connect to any REST or GraphQL backend.
